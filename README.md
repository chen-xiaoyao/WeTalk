# WeTalk
北理工计科小学期：Qt聊天室。本项目采用云服务器架构+QML前端，实现多人在线即时聊天。

我们这个项目在同一届中算做的很好的了，之所以没有做到最好，是因为有两个技术点没有实现：

1. 服务器多线程并发
2. 文件传输

实在是没时间了，如果有后来者可以读懂我们这个项目，并且完善，相信可以拿到最好的成绩，没有之一。
